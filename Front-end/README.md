## Requisitos Previos
- Node.js (versión 14.0.0 o superior) https://nodejs.org/
- npm

## Intalación y configuración

### 1. gitclone del repositorio:
```bash
git clone --branch Implementacion-de-usuarios https://github.com/Freddymb7/Proyecto-Taller-Integrador.git
```

### 2. Navegar al Proyecto:
```bash
cd [nombre-del-repo]
cd [Front-end]
```

### 3. Dependencias:

Ejecutar desde la carpeta de front-end
```bash
npm install json-server express axios react-router-dom react react-dom
npm install
```

### 4. Servidor de Fakeapi
```bash
npm run fakeapi
```

### 5. Servidor de desarrollo:
   
Ejecutar desde una nueva terminal en la misma carpeta del proyecto 
```bash
npm run dev
```
